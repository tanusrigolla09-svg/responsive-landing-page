# Responsive Landing Page
This is a simple responsive landing page built using HTML5 and CSS3.
## Features
- Responsive navigation bar
- Hero section with call-to-action button
- Features section using CSS Grid
- About and Contact sections
- Smooth scrolling
- Mobile responsiveness using media queries
## Technologies Used
- HTML5
- CSS3
- Flexbox
- Grid
- Media Queries
## Outcome
This project demonstrates understanding of responsive web design principles.
